[Terminal] Tạo Alias cho lệnh trên MacOS

 Jun 6, 2016
Để tạo được tên thay thế (alias) cho các lệnh trên Mac khá đơn giản. Ta chỉ cần thêm các lệnh alias vào file .bash_profile tại thư mục home là OK. Trường hợp máy chưa có file .bash_profile thì ta có thể tạo mới file này là OK. Lệnh tạo alias: alias YOUR_ALIAS='YOUR_COMMAND' Ví dụ ta tạo alias cho lệnh react-native với tên là rn như sau: alias rn='react-native' Vì ta chỉ tạo alias chứ không thay đổi câu lệnh nên là vẫn có thể sài được lệnh cũ sau khi đã được tạo alias.
